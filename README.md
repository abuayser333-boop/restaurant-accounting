# 🍽️ نظام محاسبة المطاعم

> برنامج محاسبي متكامل ومتخصص لإدارة المطاعم — مبني بلغة Python مع واجهة سطر الأوامر وقاعدة بيانات SQLite.

---

## ✨ الميزات الرئيسية

- **📄 إدارة الفواتير**: إنشاء وعرض وحذف فواتير المبيعات مع تفاصيل الطلبات
- **💸 إدارة المصروفات**: تسجيل وتصنيف وتتبع جميع مصروفات المطعم
- **📦 إدارة المخزون**: متابعة المواد الخام والكميات والتنبيه عند النفاد
- **📊 التقارير المالية**: تقارير يومية وشهرية للأرباح والخسائر والمبيعات
- **🗄️ قاعدة بيانات محلية**: استخدام SQLite بدون الحاجة لإعداد خادم

---

## 🗂️ هيكل المشروع

```
restaurant-accounting/
├── main.py              # نقطة الدخول الرئيسية
├── database.py          # قاعدة البيانات SQLite
├── models.py            # نماذج البيانات
├── modules/
│   ├── __init__.py
│   ├── invoices.py      # وحدة الفواتير
│   ├── expenses.py      # وحدة المصروفات
│   ├── inventory.py     # وحدة المخزون
│   └── reports.py       # وحدة التقارير
├── requirements.txt
└── README.md
```

## 🚀 طريقة التشغيل

```bash
pip install -r requirements.txt

```
